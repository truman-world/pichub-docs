# 图片处理

PicHub 在上传图片时可自动进行压缩、格式转换、缩略图生成和水印添加。

## 压缩

上传时自动压缩图片，减少存储空间和带宽消耗。

### 配置方式

在后台管理 → 系统设置 → 图片处理中配置：

- 压缩质量（1-100，默认 70）
- 是否启用自动压缩

用户在上传界面也可手动调节压缩质量。

## 格式转换

支持将上传的图片自动转换为 WebP 格式，通常能减少 30-50% 的文件体积。

在后台管理 → 系统设置中启用：

- 自动转换为 WebP
- 保留原格式副本（可选）

## 缩略图

上传成功后自动生成缩略图，用于列表展示和预览。缩略图尺寸在后台可配置。

## 水印

支持为上传的图片自动添加水印。

### 水印类型

- 文字水印：自定义文字内容、字体大小、颜色、位置
- 图片水印：上传水印图片，设置透明度和位置

### 水印位置

支持 9 个位置：左上、上中、右上、左中、居中、右中、左下、下中、右下。

### 配置方式

后台管理 → 系统设置 → 水印设置：

1. 选择水印类型（文字/图片）
2. 设置水印内容和样式
3. 选择水印位置
4. 设置透明度
5. 启用水印

## 处理顺序

上传时的图片处理按以下顺序执行：

1. 内容审核（如已启用）
2. 压缩
3. 格式转换（WebP）
4. 生成缩略图
5. 添加水印
6. 存储到目标驱动
