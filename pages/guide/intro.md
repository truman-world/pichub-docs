# 🌈 什么是 PicHub?

**PicHub** 是一款现代化、高性能的开源图片管理与托管系统。它专为个人摄影师、团队协作和企业级图像资产管理而设计。

不仅是一个图床，更是一个完整的数字资产管理解决方案 (DAM)。

## ✨ 核心特性

- **🎨 极致 UI 设计**: 采用 "Aurora" 极光风格与玻璃拟态 (Glassmorphism) 设计，提供令人惊艳的视觉体验。
- **☁️多存储策略**: 支持本地存储以及 AWS S3、阿里云 OSS、腾讯云 COS 等兼容 S3 协议的对象存储。
- **👥 多租户与权限**: 完善的角色管理（管理员、会员、访客），支持私有相册与公开分享。
- **🚀 零配置安装**: 内置图形化安装向导，三步即可完成环境检测与数据库部署。
- **🔒 隐私与安全**: 细粒度的上传策略控制，支持临时链接、防盗链设置。
- **🔌 API 优先**: 提供完整的 RESTful API，方便集成到其他应用或开发第三方客户端。
- **📦 商业化支持**: 内置商业授权系统，支持由开源版无缝升级至 Pro/Enterprise 版。

## 🛠️ 技术栈

PicHub 基于现代主流技术栈构建，确保稳定、高效且易于维护：

- **后端**: Laravel 11.x (PHP 8.2+)
- **前端**: Blade + Tailwind CSS + Alpine.js
- **管理面板**: FilamentPHP v3
- **数据库**: MySQL 8.0+
