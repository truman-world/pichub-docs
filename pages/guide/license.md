PicHub 基于 MIT 协议开源，所有核心功能完全免费。商业授权面向有额外需求的企业和个人用户。

## 开源 vs 商业授权

| 项目 | MIT 开源版 | 商业授权版（¥99） |
|---|---|---|
| 全部功能 | ✅ | ✅ |
| 商业环境使用 | ✅ | ✅ |
| 底部版权信息 | 必须保留 | **可去除** |
| 功能更新 | 社区版本 | **优先获取** |
| 技术支持 | GitHub Issues | **1 年邮件支持** |
| 授权域名 | — | **1 个主域名** |

> [!NOTE]
> 开源版用户在任何场景下均可免费使用 PicHub 的全部功能，唯一要求是保留页面底部的版权链接。

## 购买授权

访问 [pichub.app/license](https://pichub.app/license) 完成购买。支持支付宝、微信支付。

购买后你将收到一个格式为 `PICHUB-XXXX-XXXX-XXXX` 的授权码。

## 激活步骤

### 1. 配置授权服务器地址

确保 `.env` 中配置了授权服务器 URL（通常无需修改）：

```env
LICENSE_API_URL=https://license.pichub.app/api/verify
```

### 2. 在后台激活

登录 Filament 管理面板 → **系统设置** → **授权管理**：

1. 输入你的授权码
2. 点击「激活授权」
3. 系统会自动绑定当前域名并验证授权

激活成功后，你将看到授权状态变为「已激活」，底部版权信息将自动隐藏。

### 3. 验证授权

你可以随时在 [license.pichub.app/verify](https://license.pichub.app/verify) 查询授权码状态。

## 常见问题

### 授权码可以换域名吗？

可以。请联系 [support@pichub.app](mailto:support@pichub.app) 申请域名迁移。

### 一个授权码能用在多个站点吗？

Pro 授权码限 1 个主域名。如需多域名支持，请购买多个授权码。

### 授权过期后会怎样？

授权为**一次买断终身可用**，不会过期。1 年邮件技术支持到期后，功能不受影响。

### 不购买授权会影响功能吗？

不会。PicHub 的所有功能对开源用户完全开放，无任何功能限制。
